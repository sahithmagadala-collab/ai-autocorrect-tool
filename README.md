# AI Autocorrect Tool

An AI-powered autocorrect web application built using Python, NLP, and Streamlit.

## Features
- Spell correction
- NLP-based text processing
- Interactive web UI
- Real-time correction

## Technologies Used
- Python
- Streamlit
- pyspellchecker
- NLP

## Installation

pip install -r requirements.txt

## Run Project

streamlit run app.py
